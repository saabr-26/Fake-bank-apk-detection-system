# 🏦 Fake Bank APK Detection System (Cybersecurity Project)

This project is designed to detect **malicious or fake banking APK files** using analysis techniques. It helps in identifying potentially harmful Android applications that mimic legitimate banking apps.

This project was developed as part of my **B.Tech coursework in Cybersecurity**, focusing on **mobile security, malware detection, and safe application analysis**.

---

## 🚀 Features

- Upload and analyze APK files  
- Detect suspicious or fake banking applications  
- Simple web-based interface for interaction  
- Backend API for processing APK files  
- Supports testing with sample APKs  

---

## 🧠 Objective

- Identify fake or malicious banking apps  
- Understand APK structure and behavior  
- Explore real-world cybersecurity threats  
- Build a practical security-focused system  

---

## 🛠️ Tech Stack

- Python  
- Flask (Backend API)  
- HTML, CSS, JavaScript (Frontend)  
- Docker (for service simulation)
